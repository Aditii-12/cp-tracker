# CP Tracker

CP Tracker is a data-driven tool that analyzes competitive programming performance using public platform APIs. It helps users track contest ratings, problem-solving trends, and topic-wise strengths and weaknesses over time.

## Features
- Fetches user data from Codeforces public API  
- Tracks contest rating history  
- Analyzes solved problems by topic and difficulty  
- Provides insights into performance trends  

## Tech Stack
- Python  
- Codeforces Public API  
- Pandas  
- Streamlit (planned)
